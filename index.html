Chas Academy Fullstack Java Project

This project is a Progressive Web App (PWA) built with Vite that provides a fast and responsive interface for exploring data from the Star Wars universe.

About the Project

This project was developed as a frontend application that consumes the Star Wars API (SWAPI). It presents characters, planets, species, vehicles, and other resources in a structured, searchable, and filterable format.

Features

  • Search within categories
  • Dynamic filtering and sorting
  • Responsive layout
  • Offline support via PWA
  • Installable as a mobile/desktop app

Technology Stack

  • Vite
  • Vanilla JavaScript (ES Modules)
  • Vite-PWA-Plugin
  • Star Wars API (SWAPI)

Purpose

The goal of this project was to demonstrate modern frontend development principles including modular architecture, semantic HTML, responsive design, and progressive enhancement and an ability to implement such under Agile Development
